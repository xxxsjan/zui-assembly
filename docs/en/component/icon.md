# Icon

Zui assembly recommends using icons as the icon library.

```
$ pnpm install @vicons/ionicons5
```

### Icon Props

| Parameter | Type             | Optional | Explain    | Default value |
| --------- | ---------------- | -------- | ---------- | ------------- |
| color     | string           | -        | Icon Color | -             |
